# Campus Hub TV

## Short Description

Android TV signage shell with QR setup and web-powered campus displays.

## Full Description

Campus Hub TV turns Android TV and Google TV devices into managed digital signage screens for campus, student life, and community announcements.

This Android TV app is a thin display shell built for unattended screens. It loads a Campus Hub display in a full-screen WebView, supports direct local setup from a phone, and keeps the signage runtime simple enough for kiosk-style deployments.

Use Campus Hub TV to:

- launch a campus display on Android TV
- configure the screen from a phone on the same local network
- pair through a QR code and short setup code
- apply a live display URL or JSON display config
- reload, reset, and identify the screen without attaching a keyboard

Why Android TV:

- Android TV works well as a dedicated signage runtime
- the app can be locked down in kiosk deployments
- the display experience stays web-powered, so widget rendering remains consistent with the main Campus Hub product

Setup flow:

1. Open Campus Hub TV on the Android TV device.
2. Press Menu to open the setup screen.
3. Scan the QR code from a phone on the same Wi-Fi or Ethernet network.
4. Apply a display URL or config and return to the live screen.

Campus Hub TV is designed for:

- campus buildings
- student centers
- residence halls
- event signage
- departmental displays

This release focuses on Android TV and Google TV as the supported managed TV deployment path for Campus Hub.
