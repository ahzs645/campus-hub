# Play Console Package

Prepared Android TV release assets and copy for the first Play submission.

## Generated graphics

- Android TV banner graphic: `graphics/android-tv-banner-1280x720.png`
- TV screenshot, display: `screenshots/tv-display-1280x720.png`
- TV screenshot, setup: `screenshots/tv-setup-1280x720.png`

## App bundle

- Release bundle: `android/app/build/outputs/bundle/release/app-release.aab`

Note: the bundle currently builds successfully, but the Android project still needs a production signing config before Play submission. The current Gradle release build is still using the debug keystore.

## Listing copy

- Main listing copy: `store-listing.en-US.md`
- Reviewer notes: `reviewer-notes.md`

## Asset generation

- Launcher icons and Android TV banner resources are generated by `scripts/generate_tv_assets.py`

## Upload checklist

1. Replace the debug release signing config with your production keystore or Play App Signing flow.
2. Upload `android/app/build/outputs/bundle/release/app-release.aab`.
3. Upload `graphics/android-tv-banner-1280x720.png`.
4. Upload the TV screenshots from `screenshots/`.
5. Paste the copy from `store-listing.en-US.md`.
6. Paste the review instructions from `reviewer-notes.md`.
